# Catálogo A&D - Ventas y Suministros

Catálogo de productos online para Ventas y Suministros A&D.

## 📋 Características

- ✅ Búsqueda de productos por nombre o descripción
- ✅ Filtrado por categoría
- ✅ Ordenamiento por precio
- ✅ Visualización de disponibilidad
- ✅ Diseño responsivo (funciona en móvil, tablet y desktop)
- ✅ Interfaz moderna y atractiva

## 📁 Estructura de archivos

- `index.html` - Página principal del catálogo
- `styles.css` - Estilos y diseño
- `productos.js` - Base de datos de productos
- `catalogo.js` - Lógica de filtros y búsqueda

## 🚀 Cómo agregar productos

Edita el archivo `productos.js` y agrega nuevos productos al array. Ejemplo:

```javascript
{
    id: 9,
    nombre: "Tu producto",
    categoria: "Tu categoría",
    precio: 10.00,
    descripcion: "Descripción del producto",
    emoji: "🎯",
    disponible: true
}
```

## 📱 Vista previa

Tu catálogo estará disponible en: `https://ventasysuministrosayd.github.io`

## 💡 Personalización

Puedes cambiar:
- Los colores en `styles.css` (líneas con #667eea y #764ba2)
- El nombre y descripción en `index.html`
- Los emojis en `productos.js`
- Las categorías según tus productos
